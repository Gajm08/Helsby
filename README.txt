Helsby Walkthrough (Phone-friendly)
================================

How to view:
- Upload this folder to a static host (GitHub Pages works great), or
- Open index.html with a local web server.

Controls (phone):
- Swipe left/right to change stops (rooms).
- Pinch to zoom, drag to pan.
- Double-tap to reset zoom.
- Tap 'Floorplan' to view the plan.

Files:
- index.html
- manifest.json
- assets/ (images, including floorplan.jpg)
